# About id-generator
### Usage
``` config ``` currently has the following options :  
1. ``` length ``` a number value to specify the length of the ID.
2. ``` uppercase ``` if set to true, the generated id will include lowercase letters (a-z). DEFAULT is true
3. ``` lowercase ``` if set to true, the generated id will include uppercase letters (A-Z). DEFAULT is false
4. ``` numbers ``` if set to true, the generated id will include numbers (0-9). DEFAULT is true.
5. ``` prefix ``` add a string to the end of the ID.
6. ``` suffix ``` add a string to the beginning of the ID.
